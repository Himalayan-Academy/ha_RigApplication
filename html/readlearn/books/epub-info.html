How to Install and Read Ebook Files

iPad, iPhone or iPod touch

Using iBooks
  1. Connect your device to your computer
  2. Open iTunes
  3. Drag the ePub files to the Books icon in the left sidebar, under Library
  4. Select your device in the left sidebar, under Devices
  5. In the main screen, choose the Books tab at the top
  6. Make sure that Sync Books is checked
  7. Either click on the "All Books" option or select the books that you want to sync to your device
  8. Sync your device
Using Stanza
  1. Connect your device to your computer
  2. Open iTunes
  3. Select your device in the left sidebar, under Devices
  4. In the main screen, choose the Apps tab at the top
  5. Scroll down to the File Sharing area and select Stanza
  6. Drag your ePub files to the Stanza Documents area
  7. Sync your device

Mac OS or Windows

Using Adobe Digital Editions
  1. Install Adobe Digital Editions if you have not done so already
  2. In Adobe Digital Editions's top left corner, click on Library, then Add to Library
  3. Choose your ePub file

Note: If Adobe Digital Editions is unable to read some of the ebook files, be sure that you have the latest version of the software installed.

Using Calibre

(Also available for Linux.)

  1. Download and install Calibre if you have not done so already (Calibre will request you to choose a location for your ebook library)
  2. Drop ebooks or PDFs on the Calibre window and they will be copied to your library
  3. If an iOS device is connected, Calibre can also read ebooks from the device.

Kindle (Mobi format)

  1. Connect the Kindle to your computer and wait for the Kindle to appear as a drive
  2. Copy the Kindle file (with a .mobi extension) to the Kindle's Documents folder
Using Barnes & Noble E-Reader
  1. Install eReader for Mac or Nook for PC if you have not done so already
  2. Open any ePub file with the Barnes & Noble application. Note: This application does not provide a library service, so you may organize your ebooks on your computer any way you like.

Nook

  1. Connect the Nook to your computer and wait for the Nook to appear as a drive
  2. Copy the ePub file (be sure it is the "_bn.epub" version) to the Nook's My Documents folder
  3. To view it in the Nook, go to My Library
  4. Tap View My Documents
  5. Tap Check for New Content
  6. Select the book

Sony Reader

  1. Install the Reader Library application if you have not done so already
  2. Connect the Sony Reader to your computer
  3. Copy the ePub file to the Reader's My Documents folder
  4. Sync your reader
